# レイトレ徒競走 〜交差エンジン編〜

## ルール
- 入力画像に対するAO画像を出力してください
- リファレンス画像から一定以内のMSEに抑えてください。
- 入力されるメッシュは次のような特徴があります。
-- Inf/NaNが入力される可能性があります。その場合はそれを含む面は無視してください。
-- 32bitインデックスは超えません
- ビルド済みのdllを提出してください

## 禁止事項
- GPUは一切使用しないでください
- Embreeなどサードパーティの交差判定ライブラリは使用しないでください。

## 実装に関して
- rayRun()関数は何度も呼び出されるので初期化が必要な場合はdllmain()か初回のrayRun()のみ行うようにしてもかまいません。
- 本番当日のテストベッドはimageポインタを見て観戦するようなものを用意します。途中結果もできるだけimageに書き込んでください。
